System Integrations Guide

1. Email Notifications (Webhook)

To send automatic emails (like inventory checks or password resets), you need to set up a webhook receiving endpoint, such as on Zapier or Make.com.

  1. Create a new Scenario/Zap.
  2. Choose "Webhooks" as the trigger and select "Catch Hook".
  3. Copy the URL provided and paste it into the Webhook URL field in the settings.
  4. Add an action step like "Gmail - Send an Email" or "Office 365 - Send an Email".
  5. Map the incoming webhook fields to the email action (To, Subject, HTML Body).

2. Cloud Synchronization (Supabase)

Supabase allows your asset data to be stored securely in the cloud, instead of permanently bound to this computer. This ensures that when you email Custodian Portal Links, they actually function correctly off-network.

  1. Sign up for free at Supabase.com.
  2. Create a New Project. Wait for the database to provision.
  3. Go to the Settings (gear icon) -> API section.
  4. Copy the Project URL and Project API Keys (anon public).
  5. Paste these values into the Settings tab of this application.
  6. The system will automatically create an `app_state` table and sync your data securely.
